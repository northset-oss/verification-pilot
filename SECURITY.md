# Security policy

## Supported version

The current `main` branch is supported. Historical signed bundles are immutable records and are not silently replaced.

## Reporting a vulnerability

Please report vulnerabilities privately through GitHub's security-advisory interface for this
repository or by email to `oss@northset.ai`. Do not include tokens, private keys, credentials,
personal data, or exploit payloads in a public issue.

Include the affected file or workflow, the security boundary involved, reproduction steps that do not expose secrets, and the expected impact. Northset will acknowledge the report, investigate it, and publish a correction or release note when appropriate. A report does not authorize testing against third-party systems.
