# northset-oss

Northset OSS Run Records — public home of the Northset Maintainer Verification Pilot.

This repository holds the open `mission.json` receipt schema, the validator CLI, run-record
bundle tooling, and the public mission ledger. Every run record here is evidence of declared
execution metadata and artifacts. It is not proof of code quality, security, maintainer
approval, or production readiness.

Status: pre-launch scaffold (2026-07-09). Nothing here is published or announced yet.
